# ゲームアイディア: アンビバレント・オペラ

## コンセプト
「下手さ」と「愛おしさ」という本来矛盾するはずの2つの評価が同時に高まる、
アンビバレント(両価的)な瞬間を競うパーティーゲーム。
実際は歌が下手なのに愛され続けたオペラ歌手フローレンス・フォスター・ジェンキンスの
逸話をモチーフにしている。

## 形式
- ジャンル: 即興パフォーマンス系パーティーゲーム
- プレイ人数: 4〜8人(パフォーマー1人+観客役の他プレイヤー)
- プレイ時間: 1ラウンド3〜5分 × 参加人数分ラウンド

## 遊び方(ルール)
1. 各ラウンド、1人が「パフォーマー」役になり、お題カード(例:「オペラの一節」
   「情熱的なポエム」「感動のスピーチ」)を引く。
2. パフォーマーは、お題をわざと下手に、しかし全力で・愛嬌たっぷりに演じる
   (フローレンス・フォスター・ジェンキンス風に、本人は上手いつもりで熱演する)。
3. 観客役のプレイヤー全員が、以下2枚の評価カードをそれぞれ同時に伏せて提出する。
   - 「下手さ」カード: 1〜5点で下手さを評価
   - 「愛おしさ」カード: 1〜5点で愛おしさ・魅力を評価
4. 2種類のカードを同時に公開し、各観客の「下手さ×愛おしさ」の積を合計してパフォーマーの得点とする。
   両方が高いほど(矛盾した評価が同時に成立するほど)高得点になる。
5. 全員がパフォーマー役を一巡したら終了。

## 勝利条件
全ラウンド終了時点で、パフォーマーとして獲得した得点の合計が最も高いプレイヤーが勝利。
